# ft_printf

## Description

ft_printf is a project from the 42 curriculum.

The goal of this project is to recreate a simplified version of the standard C `printf` function.  
The function must handle a variable number of arguments and print them according to format specifiers.

Prototype:

```c
int ft_printf(const char *format, ...);
